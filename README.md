# Active Directory Home Lab

## Project Status

In Progress

## Project Overview

This project documents the setup of a basic Windows Active Directory lab using VMware Workstation. The lab will include a Windows Server domain controller, a Windows client machine, Active Directory Domain Services, organizational units, users, security groups, and basic Group Policy configuration.

The goal of this lab is to build hands-on experience with enterprise identity management, Windows Server administration, domain environments, and security-focused documentation.

## Lab Goals

- Install and configure a Windows Server virtual machine
- Install the Active Directory Domain Services role
- Promote the server to a domain controller
- Create a new domain
- Create organizational units
- Create test users and security groups
- Configure a basic Group Policy Object
- Join a Windows client VM to the domain
- Document the setup with screenshots

## Planned Lab Environment

| System | Purpose |
|---|---|
| Windows Server | Domain Controller |
